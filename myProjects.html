Project 3 - Diabetes Regression
About the Project

This project involves building a linear regression model to find the line of best fit. First, data is visually analyzed, and a correlation matrix is created to identify the highest correlated attribute. The least correlated attributes are dropped, and the highest correlation is used as a key feature for the model. The data is then cleaned, and the highest correlated attribute (x) and target data (y) are plotted. A linear regression model is trained using .fit(), with the input data reshaped into a 2D format to be compatible with sklearn. The model predicts the line of best fit, which is then plotted alongside the data with appropriate labels.

Technical: Matplotlib, pandas, scikit-learn, numpy, LinearRegression

Project 5 - Diabetes Classification Revisited
About the Project

In this project, I built a Decision Tree model using both the Gini index and Entropy criteria to compare their effectiveness. I started by wrangling the dataâ€”gathering it, removing duplicates, and handling null values. Then, I split the dataset into features (X) and the target variable (y), followed by partitioning them into training and testing sets.

Next, I created and trained separate Decision Tree models for both Gini and Entropy using .fit() with the respective training data. I visualized the models in a single figure with one row and two columns, labeled according to the feature names and target values ("Yes" for 1 and "No" for 0).

Project 7 - Titanic Classification
About the Project

In this project, I wrangled the data to include only relevant features (age, pclass, sex) and handled duplicates, nulls, and one-hot encoding.

I applied standard scaling to the training and test sets to avoid data leakage, then set up a KNN model. Using cross-validation across a range of k values, I determined the optimal number of neighbors for the model.

I trained and evaluated the final KNN model, plotted accuracy versus number of neighbors, and made predictions for new data, including a specific example of a 31-year-old female passenger in 2nd class.

Technical: Matplotlib, pandas, numpy, scikit-learn, KNeighborsClassifier, pipeline, train_test_split, StandardScaler, accuracy_score, classification_report, cross_val_score

Project 8 - Wine Quality Clustering
About the Project

In this project, I prepared the red wine dataset by removing duplicates, nulls, and the wine index, and separated the quality column for later evaluation. I normalized the data and used K-Means clustering, finding the optimal number of clusters (k) using inertia and silhouette scores.

After training the model with the best k, I assigned cluster labels, evaluated clustering performance with a silhouette score, and compared clusters to actual wine quality using a contingency table. Questions related to the project were also answered.

Technical: Matplotlib, pandas, scikit-learn, Normalizer, train_test_split, KMeans, silhouette_score
