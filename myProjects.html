Project 2 - Avocados
About the Project

This project uses the avocado.csv dataset to create three plots. The first plot shows raw data, with AveragePrice vs Time and Total Volume vs Time. The second plot aggregates the data by grouping AveragePrice and Total Volume by Date, using a summing function to provide more valuable insights. The AveragePrice is adjusted for incorrect processing. The third plot applies the .rolling() method to the aggregated data to smooth out trends and highlight long-term patterns. A for loop is used to modify the graph for better representation.

Technical: Matplotlib, pandas, datetime

Project 3 - Diabetes Regression
About the Project

This project involves building a linear regression model to find the line of best fit. First, data is visually analyzed, and a correlation matrix is created to identify the highest correlated attribute. The least correlated attributes are dropped, and the highest correlation is used as a key feature for the model. The data is then cleaned, and the highest correlated attribute (x) and target data (y) are plotted. A linear regression model is trained using .fit(), with the input data reshaped into a 2D format to be compatible with sklearn. The model predicts the line of best fit, which is then plotted alongside the data with appropriate labels.

Technical: Matplotlib, pandas, skikit-learn

Project 4 - Diabetes Classification
About the Project

This project involves data wrangling steps such as gathering data, removing unusable data, creating a correlation matrix, and splitting the data into feature vectors and targets. It also includes partitioning the data into training and testing sets. For logistic regression, the model is set up, trained, and tested, with predictions made and accuracy evaluated. A set seed is used to ensure consistent training and testing data, preventing randomization in results each time. The final output is the accuracy score of the model.

Technical: Matplotlib, pandas, skikit-learn, joblib

Project 5 - Diabetes Classification Revisited
About the Project

In this project, I built a Decision Tree model using both the Gini index and Entropy criteria to compare their effectiveness. I started by wrangling the dataâ€”gathering it, removing duplicates, and handling null values. Then, I split the dataset into features (X) and the target variable (y), followed by partitioning them into training and testing sets.

Next, I created and trained separate Decision Tree models for both Gini and Entropy using .fit() with the respective training data. I visualized the models in a single figure with one row and two columns, labeled according to the feature names and target values ("Yes" for 1 and "No" for 0).

To evaluate the models, I generated predictions using .predict() on the test set and calculated their accuracy using accuracy_score from sklearn. This allowed me to compare the two criteria and determine which performed better.

Technical: Matplotlib, pandas, skikit-learn
